World Wide Web General

1. What are the most essential technologies that made the World Wide Web such a success?
The Hypertext, Hypermedia HTTP and browsers

2. How are resources (html pages, images, etc) adressed in the web?
HTTP

3. What are the general components of a url?
The general components of a url are;

  • a scheme (identifies the protocol to be used to access the resource on the Internet - it can be HTTP or HTTPS)
  • a host (the host name identifies the host that holds the resource, Host names can be followed by a port number)
  • a path (identifies the specific resource in the host, that web client wants to access)
  • a query string

    1. What is a web server?
    A Web server is a program that uses HTTP (Hypertext Transfer Protocol) to serve the files that form Web pages to users, in response to their requests, which are forwarded by their computers' HTTP clients. Dedicated computers and appliances may be referred to as Web servers as well.

    2. What does a web server?
    It save, process and delivers the web pages to clients.

    3. How can content be stored on a web server?
    On the Hypertext Transfer protocol (HTTP)

    4. How is content in webpages described?
    HTML document
